# DevCard Web

The web backup frontend for DevCard, built with [SvelteKit](https://kit.svelte.dev/).

This app provides:
- The landing page for DevCard
- Public, browser-accessible profiles with view analytics
- Cross-platform analytics source tracking

## Developing

```bash
pnpm install
pnpm dev
```

## Building

To create a production version of your app:

```bash
pnpm build
```
